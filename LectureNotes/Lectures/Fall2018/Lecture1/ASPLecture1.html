1 Gaussian Processes

Recall the following definitions:

  • •

    Covariance:

    cov⁡[X,Y]=𝔼⁢[(X-𝔼⁢[X])⁢(Y-𝔼⁢[Y])]=𝔼⁢[X⁢Y]-𝔼⁢[X]⁢𝔼⁢[Y]
  • •

    Variance:

    var⁡[X]=cov⁡[X,X]

Covariance is bilinear, so

cov⁡[a⁢X+b⁢Y,Z]=a⁢cov⁡[X,Z]+b⁢cov⁡[Y,Z],

and in particular,

var⁡[a⁢X]=a2⁢var⁡[X].

Motivation: additive noise.

Let

Recall the Central Limit Theorem, which essentially says “(well-enough behaved) additive noise makes Gaussian distributions”. That is, adding up a bunch of small things that make the same-size contribution and rescaling yields basically a Gaussian distribution.

1.1 Brownian Motion

Let Sk,n be defined as above. Let Bt(N)=1N⁢S0,⌊t⁢N⌋. Then let Bt=limN→∞⁡1N⁢S0,⌊t⁢N⌋. Then {Bt} is Brownian motion. We can visualize Brownian motion by considering the graphs yielded by the maps t↦Bt and t↦Bt(N).

For example, if we let N=25, the function t↦Bt(N) yields the following graph for t=0 to t=10, for a particular sequence {Xk}k∈ℤ≥0.

The Central Limit Theorem tells us that Bt-Bs∼N⁢(0,t-s). Additionally,

var⁡[Bt]=limN→∞⁡⌊t⁢N⌋N=t

and

cov⁡[Bs,Bt]=s

for s≤t.

Definition

A standard Brownian motion is a stochastic process {Bt}t≥0 such that

  1. [(i)]

  2. 1.

    B0=0

  3. 2.

    Bt-Bs∼N⁢(0,t-s) - that is, the variance of an increment is proportional to the time difference

  4. 3.

    Bt-Bs is indepenedent of Bv-Bu for u<v≤s<t

That is, Brownian motion is the stochastic process with independent Gaussian increments – i.e., how it moves in an interval just depends on the length of that interval, not where it is.

Example

Suppose a stream of energetic particules is absorbed by some object and the energy is slowly released from that object. Suppose at time t, the proportion of energy that remains in the object from a particle absorbed t time units ago is e-t. Assuming that the object only absorbs energy at times t=0,1,2,…, let Xt be the amount of energy absorbed at time t. Let Zn be the total energy contained in the object at time n. Then

Zn=∑k=0∞e-k⁢Xn-k.

We can scale the x and y axes – let

Z⌊t⁢N⌋=1N⁢∑k=0∞e-k/N⁢X⌊t⁢N⌋-k.

Then

Z⌊t⁢N⌋→N→∞Zt.

Note about previous example

If we start with Brownian motion and incorporate exponential decay, a more general central limit theorem applies (for more information, look up Lindeberg-Feller condition).

Definition

Random variables Z1,…,Zn (which we can think of as a random vector of length n, (Z1,…,Zn)) are jointly Gaussian if for any a1,…,an∈ℝ, we have ∑k=1nak⁢Zk∼N⁢(m,σ2) for some m, σ2. That is, these random variables are jointly Gaussian if and only if any linear combination of them is univariate Gaussian.

Definition/Notation

Suppose that the random variables (Z1,…,Zn) are jointly Gaussian. Let Z=(Z1,…,Zn). Define a vector of means μ=(μ1,…,μn), where μi=𝔼⁢[Zi]. Define the n×n covariance matrix Σ={Σi,j}, where Σi,j=cov⁡[Zi,Zj]. Then we write Z∼N⁢(μ,Σ).

Now, suppose that we have Z, μ, and Σ as in the above definition, and let a=(a1,…,an)∈ℝn.

Then we know that ∑k=1nak⁢Zk∼N⁢(m,σ2) for some m, σ2. A small amount of algebraic manipulation yields the following equations:

m=∑k=1nak⁢μk=aT⁢μ,

and

σ2=∑1≤i,j≤nai⁢Σi,j⁢aj=aT⁢Σ⁢a.

Example

We can have random variables that are marginally Gaussian (that is, each of them have Gaussian distributions themselves) but that are not jointly Gaussian. Let X∼N⁢(0,1) and Y∼N⁢(0,1) be independent random variables. Let Z=sign⁢(X)⁢|Y|. So X and Z are “almost” independent, but have the same sign as each other. Then X and Z both have Gaussian distributions, but X and Z are not jointly Gaussian.

Definition

A Gaussian process on an index set T is a collection of random variables {Xt}t∈T such that for any n∈ℤ>0, for any (t1,…,tn)∈Tn, (Xt1,…,Xtn) is jointly Gaussian. It is centered if 𝔼⁢[Xt]=0 for all t.

The distribution of a Gaussian process is determined by its mean, μ⁢(t)=𝔼⁢[Xt] and covariance (also called covariance kernel) σ2⁢(s,t)=cov⁡[Xs,Xt].

Example

Let Z∼N⁢(μ,Σ), where Z=(Z1,…,Zn). Then Z is a Gaussian process on the index set {1,2,…,n}.

Example

{Bt}t≥0, Brownian motion, is a Gaussian process on [0,∞).

1.2 Facts About Jointly Gaussian Variables and Gaussian Processes

Multivariate Gaussian density

If Z∈ℝn and Z∼N⁢(μ,Σ) (that is, Z is an n-dimensional multivariate Gaussian), then

𝔼⁢[f⁢(Z)]=∫ℝnf⁢(x)⁢1(2⁢π)n⁢det⁢(Σ)⁢exp⁡(-(x-μ)T⁢Σ-1⁢(x-μ)2)⁢𝑑x.

Linear transformations of Gaussians are Gaussian

if Z=(Z1,…,Zn)∼N⁢(μ,Σ) and A∈ℝk×n, then A⁢Z∼N⁢(A⁢μ,A⁢Σ⁢AT).

Multivariate simulation with the Cholesky

Let Σ=K⁢KT be the Cholesky decomposition of Σ, and let Z=(Z1,…,Zn) independent, each with distribution N⁢(0,1). Then if we let (X1,X2,X3)=X=K⁢Z, then X∼N(0,KIKT=Σ).

Zero correlation implies independence for Gaussians

If X,Y are jointly Gaussian, then X and Y are independent if and only if cov⁡[X,Y]=0.

Covariance kernels are positive semidefinite

Given a Gaussian process on T, where T is a measure space, we can use this to define an inner product on functions T→ℝ: Given f,g:T→ℝ, define ⟨f,g⟩σ=∑s∈T∑t∈Tf⁢(s)⁢g⁢(t)⁢σ2⁢(s,t) and ∥f∥σ=⟨f,f⟩σ≥0.

Example For Brownian motion, 𝔼⁢[Bt]=0, cov⁡[Bs,Bt]=σ2⁢(s,t)=min⁢(s,t). So, for f,g:[0,∞)→ℝ,

⟨f,g⟩σ=∫0∞∫0∞f⁢(s)⁢g⁢(t)⁢min⁢(s,t)⁢𝑑s⁢𝑑t.

Isometry

If you have a linear space V, with a symmetric positive definite inner product ⟨⋅,⋅⟩, and a countable orthonormal basis {φk}k=1∞, then you can define a centered isomorphic Gaussian process, i.e., a Gaussian process on V such that 𝔼⁢[Xt]=0 for all t∈V, and cov⁡[Xs,Xt]=⟨s,t⟩. That is, you can construct random variables and index them using V in such a way that the covariance of two random variables is equal to the inner product of their indices.

This can be done in the following way. Let {Zk}k=1∞ be independent, identically distributed (iid) variables each with distribution N⁢(0,1). For t∈V, which we can write as t=∑k=1∞⟨t,φk⟩⁢φk, define Xt=∑k=1∞⟨t,φk⟩⁢Zk∈ℝ. Because Zk are independent, and each have variance 1,

cov⁡[Zi,Zj]={1if ⁢i=j0if ⁢i≠j

and so

cov⁡[Xs,Xt] =cov⁡[∑k⟨s,φk⟩⁢Zk,∑j⟨t,φj⟩⁢Zj]
=∑j,k⟨s,φk⟩⁢⟨t,φj⟩⁢cov⁡[Zk,Zj]
=∑j⟨s,φj⟩⁢⟨t,φj⟩
=⟨s,t⟩.

Example

Suppose we wanted to construct a Gaussian process on {1,2,3} with covariance matrix

Σ=(1120121120121).

This is just a fancy way of saying we want ot construct jointly Gaussian random variables (X1,X2,X3) with covariance matrix Σ. We can do this by the Cholesky method above, but another way is as follows. Let Σ=V⁢Λ⁢VT be a diagonalization of Σ, i.e., the columns of V are the normalized eigenvectors of Σ, and Λ is diagonal with the corresponding eigenvalues. Since Σ is symmetric, V is orthogonal and the entries of Λ are real. Then let

Zi∼N⁢(0,λi),

and let

X=V⁢Z.

Then X∼N(0,VΛVT=Σ), as desired.

Example, continued

Now suppose we want to construct a Gaussian process on ℝ3 with covariance kernel given by Σ, i.e., for any v∈ℝ3, a random variable Yv such that for any u and v, we have cov⁡[Yu,Yv]=uT⁢Σ⁢v, where Σ is the covariance matrix from the previous example.

We can do this using the X above by setting

Y(u1,u2,u3)=u1⁢X1+u2⁢X2+u3⁢X3.
