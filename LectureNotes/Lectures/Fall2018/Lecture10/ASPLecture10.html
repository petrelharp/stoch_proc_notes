Stationary Distributions and Recurrence (with David Levin)
November 5 and 7

Recall: We’re interested in probability measures π which are solutions to π⁢G=0.

Assume we have a countable (or finite) state space X. gi,j represents the transition rate from i to j, and gi=∑j≠igi,j is the rate of leaving state i. Suppose maxi⁡gi=g<∞ (uniformly bounded rates).

Let Pi,j=gi,jg for j≠i. Then ∑j≠iPi,j=gig≤1. Let Pi,i=1-gig so that ∑jPi,j=1. P is a transition matrix with G=g⁢(P-I).

Note, π⁢G=0 iff π⁢P=π. In continuous time, Pt=et⁢G is the operator corresponding to moving forward t units of time. π⁢Pt=π⁢∑k=0∞tk⁢Gkk!=π since nonzero powers of G vanish under π. π⁢Pt is the distribution at time t, starting at π, so this says we will remain in π for all times t.

Existence: When is there a solution to π⁢P=π⁢?

Assume P is irreducible so that for all i,j, there exists t such that Pt⁢(i,j)>0 (meaning every state is reachable by any other state eventually.) Then there exists a solution to π⁢P=π but π need not be a finite measure.

Example: Consider the simple random walk on the integers with Pi,j=12 if j=i±1 and Pi,j=0 otherwise. The solution is π=1.

Definition: A chain with transition matrix P is called positive recurrent if π⁢P=π has a finite measure solution π. A state i is called recurrent if ℙi⁢(Return to ⁢i)=1. Otherwise, the state is called transient.

If P is irreducible, then either all states are recurrent or all states are transient.

Theorem: A state i is recurrent iff 𝔼i⁢[τi+]<∞ where τi+=min⁡{t≥1:Xt=i}. Equivalently, ℙi(τi+<∞)=1. (Obvious when the state space X is finite and irreducible.)

Example: A random walk on the non-negative integers where the state increases by one with probability p and decreases by one with probability q=1-p. (The transition from 0 to 0 takes probability q.)

p>q implies the chain is not recurrent. If q>p, then the chain might be recurrent. For k≠0 we have π⁢(k)=π⁢(k-1)⁢p+π⁢(k+1)⁢q since there are two different ways to arrive at state k. As the special case, we have π⁢(0)=π⁢(0)⁢q+π⁢(1)⁢q.

We want to solve π⁢P=π. We get π⁢(1)=pq⁢π⁢(0) and π⁢(k)=(pq)k⋅π⁢(0) as a solution. Thus, ∑kπ⁢(k)=π⁢(0)⁢(1-pq) and π⁢(0)=1-pq. We have a probability distribution when π⁢(k)=(1-pq)⁢(pq)k. Thus, positive recurrence.

Definition: A chain is aperiodic if gcd{t:Pt⁢(x,x)>0}=1. In the case of a bipartite graph, we can only return at even times, and Pt⁢(x,x)=0 if t is odd. (Not needed in continuous time.)

If the chain is positive recurrent and irreducible (with aperiodicity in the discrete time case), then there exists a unique solution π which solves π⁢P=π (equivalently π⁢G=0).

(Convergence) Theorem: The distribution starting at x after t steps converges to π as t→∞ in the sense that d⁢(Pt⁢(x,⋅),π)→0 for some metric d.

Example: d1⁢(Pt⁢(x,⋅),π)=∑y|Pt⁢(x,y)-π⁢(y)|→0 and t→∞. Thus, Pt⁢(x,y)→π⁢(y) for all x.

Now take μ to be any distribution. We have μ⁢Pt→π.

Application: Tilings of a square by dominoes. We want to select a tiling uniformly at random. The naive way to do this is list (enumerate) them all and then select one using a random number generator. In practice, this is very hard to do because there are so many possibilities. Another way might be to choose positions and orientations of dominoes in sequence until the whole space is filled, but not all tilings will occur will equal probability.

Alternatively, come up with a way to transition between random tilings such as rotating a square of two dominoes or modifying other such sub-tilings. We can construct a Markov chain with the uniform distribution as the stationary distribution π. We start by choosing the starting position according to any distribution μ and then proceed by running the chain for sufficiently large t. The closer μ is to be uniform, the faster the convergence, but μ could theoretically be very far from uniform.

Coupling: Let Xt be a chain started from x, and let Y~t be a chain started from π. Let τ=inf(t>0:Xt=Y~t). When the state space X is finite, then τ<∞. Define Yt=Y~t for t≤τ and Yt=Xt for t≥τ.

Check: Yt is a Markov chain with transition matrix P still started with π (comes from Y~t), and Yt always has distribution π.

|Pt⁢(x,z)-π⁢(z)| =|ℙ(Xt=z)-ℙ(Yt=z)|
=|ℙ(Xt=z,τ≤t)+ℙ(Xt=z,τ>t)-ℙ(Yt=z,τ≤t)-ℙ(Yt=z,τ>t)|
=|ℙ(Xt=z,τ>t)-ℙ(Yt=z,τ>t)|
≤ℙ(Xt=z,τ>t)+ℙ(Yt=z,τ>t)
≤2ℙ(τ>t)

Using ℙ(τ<∞)=1, we have 2ℙ(τ>t)→0 and thus d1⁢(Pt⁢(x,⋅),π)=∑z|Pt⁢(x,z)-π⁢(z)|→0 as t→∞.

In general, solving for the stationary distribution takes work.

Definition: P (or G) is reversible with respect to π if π⁢(x)⁢P⁢(x,y)=π⁢(y)⁢P⁢(y,x) (or π⁢(x)⁢G⁢(x,y)=π⁢(y)⁢G⁢(y,x)). These are called the detailed balance equations which say the ”flow” from x to y is the same as from y to x.

Example: Weighted random walk with P⁢(x,y)=wx,ywx where wx=∑zwx,z. P is reversible with π⁢(x)=wx/(∑zwz).

If P is reversible, then P need not be symmetric but can be diagonalized. A⁢(x,y)=π⁢(x)/π⁢(y)⁢P⁢(x,y) is symmetric. By the Spectral Theorem, A has real eigenvalues λ1>⋯>λn and the eigenfunctions φ1,…,φn form an orthonormal basis for ℝn.

A=Dπ1/2⁢P⁢Dπ-1/2 where Dπ is diagonal with Dπ⁢(i,i)=π⁢(i). fj=Dπ1/2⁢φj are eigenfunctions for P.

Define <f,g>π=∑xf(x)g(x)π(x), then {fj} are orthonormal with respect to this inner product. Pt⁢(x,y)=∑jfj⁢(y)⁢λjt⁢fj⁢(x)⁢π⁢(y).

|Pt⁢(x,y)π⁢(y)-1|≤∑j=2n|fj⁢(x)⁢fj⁢(y)|⁢λjt

where λjt’s tell us how fast the convergence is.
