Recall that G is the generating matrix for our continuous time Markov process. Moving forward we will need the following useful fact

Fact 1.
𝑠𝑝𝑒𝑐⁢(G)⊂(-∞,0]

In other words, if G⁢f=λ⁢f for some λ∈R then λ≤0.

Proof.

Suppose that f is an eigenfunction of G. Then

𝔼x⁢[f⁢(Xt)] =eλ⁢t⁢f⁢(X)

It follows that

𝔼⁢[G⁢f⁢(Xt)]=λ⁢𝔼x⁢[f⁢(Xt)]

Since f⁢(x) is bounded, so too is the right handside of the above inequality. Therefore, λ≤0. ∎

The inclusion of 0 is purposeful, as G⁢1=0, ie atleast one of G’s eigenvalues is zero.

We now return to considering how quickly the distribution of a Markov process converges to the stationary distribution. Let ν be an arbitrary probability distribution, then

ℙν{Xt=0}=νet⁢G

Letting ψj=Re⁢(ϕj⁢(ωk))/cj, if G=∑λj⁢ϕj⁢ψjT where ϕj⁢ψkT=δk⁢j we know that ϕ1=1 implies that ψ1=π. Putting this together lets us consider the following sum

∑xν⁢(x)⁢π⁢(x)+exponential decay

The slowest bit of this decay is given by et⁢λd⁢(ν⁢ϕ2)⁢ψ2T⁢(x). Therefore

ν⁢et⁢G→π⁢at speed ⁢λ2

Now that we’ve discussed mixing times, we move on to a couple of examples.

Question.
  1. 1.

    Let (Xt) be a CTMC on {1,2,…,n}, with Gi⁢i=-(1-1/n) and Gi⁢j=1/n for i≠j. Define

    τk=inf(t≥0|Xt=k)

    and let T=max⁢(τ1,…,τn). What can we say about T?

  2. 2.

    Rain falls at 1 drop per second on a region of roses subdivided into n regions. If T= first time all n pieces have been hit  then why is the time between drops exponential?

  3. 3.

    Let Hk be the time of the first drop on piece k. This is exponentially distributed with rate 1 on n. What order should Tn=max⁢(Hk) have? Is it atleast order n?

Let’s address the second question first, since it is just the first problem under a different guise. Naturally, the total number of drops is PPP on ℝ. So the probability that there are no drops in a region after time t is exponentially distributed with rate equal to 1.

For the final question, it is worth computing ℙ(Tn>t)

ℙ(Tn>t) =1-∏ℙ(Hk≤t)
=1-∏(1-e-t/n
=1-(1-e-t/n)n

If we set t=n⁢(s+log⁡n), then e-t/n=e-s. So ℙ(Tn>n(s+log(n))=ℙ(Tn/n-log(n)>s). Of course, this goes to 1-e-e-s as n→∞. Necessarily

Tn∼n⁢log⁡(n)+n⁢X

where X follows the Gumbel distribution.

More generally if Zn⁢(t)=#⁢{k:Hk>t}, then the expected value of Zn⁢(t) is n⁢e-t/n. For 𝔼⁢(Zn⁢(t))=1 we need t=n⁢log⁡(n), thus confirming that Tn is of the order n⁢log⁡(n).

Even more generally, if Y1,…,Yn are iid with the same distribution Y and Mn=max⁢(Y1,…,Yn), then defining f(t)=P(Y>t) tells us that Mn is of order X where n⁢f⁢(x)=1. ∎
