Example 1.1.

Motivating example: Moments of the Poisson

Let (Nt)t≥0 be a Poisson counting process where N∼PPP⁡(constant rate 1) on [0,∞). The fact that (Nt)t≥0 is a Poisson counting process means that Nt∈ℤ and Nt “jumps by 1 unit at rate 1”, i.e. Nt=N⁢([0,t])+N0. Note that writing N∼PPP⁡(constant rate 1) is the same as writing N∼PPP⁡(λ), where λ is the Lebesgue measure.

We want to answer the question: What is 𝔼⁢[Ntk]? First we define

𝔼x[f(Nt)]:=𝔼[f(Nt)|N0=x]

We see that

𝔼x⁢[f⁢(Nt)]=f⁢(x)+∫0tdd⁢s⁢𝔼x⁢[f⁢(Ns)]⁢𝑑s =f⁢(x)+∫0tlimϵ→0⁡1ϵ⁢𝔼x⁢[f⁢(Ns+ϵ)-f⁢(Ns)]⁢d⁢s (1)
=f⁢(x)+∫0tlimϵ→0⁡1ϵ⁢𝔼x⁢[𝔼s⁢[f⁢(Ns+ϵ)-f⁢(Ns)|Ns]]⁢d⁢s (2)

This motivates the following definition.

Definition 1.2.

Let

(G⁢f)⁢(y):=limϵ→0⁡1ϵ⁢𝔼y⁢[f⁢(Nϵ)-f⁢(y)]

G is called the generator for Nt.

Since Nϵ-N0∼Poisson⁡(ϵ),

G⁢f⁢(x) =limϵ→0⁡1ϵ⁢𝔼x⁢[f⁢(Nϵ)-f⁢(x)]
=limϵ→01ϵ𝔼[f(Nϵ)-f(x)|N0=x]
=limϵ→01ϵ(ℙ{Nϵ-N0=0}(f(x)-f(x))+ℙ{Nϵ-N0=1}(f(x+1)-f(x))
+ℙ{Nϵ-N0≥2}𝔼[f(Nϵ)-f(x)|Nϵ-N0≥2])
=limϵ→01ϵ(e-ϵ⋅ϵ⋅(f(x+1)-f(x))+∑n≥2e-ϵ⁢ϵnn!𝔼[f(Nϵ)-f(x)|Nϵ-N0≥2])
=limϵ→0⁡1ϵ⁢(e-ϵ⋅ϵ⋅(f⁢(x+1)-f⁢(x))+𝒪⁢(ϵ2))
=f⁢(x+1)-f⁢(x)

So G⁢f⁢(x)=f⁢(x+1)-f⁢(x) is the generator for the constant rate 1 Poisson process.

By the Markov property,

limϵ→0⁡1ϵ⁢𝔼x⁢[𝔼s⁢[f⁢(Ns+ϵ)-f⁢(Ns)|Ns]]=𝔼x⁢[G⁢f⁢(Ns)]

It follows from this and (2) that

𝔼x⁢[f⁢(Nt)]=f⁢(x)+∫0t𝔼x⁢[f⁢(Ns+1)-f⁢(Ns)]⁢𝑑s (3)

Next we need the following notation. Let

(x)m:=x⁢(x-1)⁢⋯⁢(x-m+1)

Observe that

(x+1)m-(x)m =(x+1)⁢x⁢(x-1)⁢⋯⁢(x-m+2)-x⁢(x-1)⁢⋯⁢(x-m+1)
=((x+1)-(x-m+1))⁢(x)m-1
=m⁢(x)m-1

Now by (3),

𝔼0⁢[(Nt)m]=0+∫0t𝔼0⁢[(Ns+1)m-(Ns)m]⁢𝑑s=0+∫0tm⁢𝔼0⁢[(Ns)m-1]⁢𝑑s

and 𝔼0⁢[(Nt)0]=1.

Let

gm⁢(t):=∫0tm⁢𝔼0⁢[(Ns)m-1]⁢𝑑s

and let g0⁢(t)=1. Then gm⁢(t)=∫0tm⁢gm-1⁢(s)⁢𝑑s, so

g1⁢(t) =𝔼0⁢[Nt]=∫0t1⁢𝑑s=t
g2⁢(t) =𝔼0⁢[Nt⁢(Nt-1)]=∫0t2⁢s⁢𝑑s=t2

and by induction,

𝔼0⁢[(Nt)m]=gm⁢(t)=tm
Example 1.3.

What is the generator for Brownian motion?

Let (Bt)t≥0 be a Brownian motion, Bt+s-Bt∼Normal⁡(0,s). Then

𝔼x⁢[f⁢(Bϵ)] ≈𝔼x⁢[f⁢(x)+f′⁢(x)⁢(Bϵ-x)+12⁢f′′⁢(x)⁢(Bϵ-x)2+⋯]
=f⁢(x)+f′⁢(x)⋅0+12⁢f′′⁢(x)⋅ϵ+𝒪⁢(ϵ3/2)

So

limϵ→0⁡1ϵ⁢𝔼x⁢[f⁢(Bϵ)-f⁢(x)]=12⁢f′′⁢(x)

and thus G⁢f⁢(x)=12⁢f′′⁢(x) is the generator for standard Brownian motion.

Example 1.4.

How can we make Nt into Brownian motion? We know that:

  1. 1.

    By the Central Limit Theorem, adding up independent noise, centering and scaling gets you the Gaussian distribution.

  2. 2.

    Nt is the number of points in a PPP on [0,t], and so is a sum of a bunch of independent things.

  3. 3.

    Brownian motion started at zero has 𝔼⁢[Bt]=0, 𝔼⁢[Bt2]=t.

Nt does not have enough noise in each interval to be Brownian. So consider NM⁢t. Since 𝔼⁢[NM⁢t]=M⁢t, we subtract M⁢t, and then divide by M to get the correct variance. Thus we define

Xt(M)=NM⁢t-M⁢tM

Let GM denote the generator of Xt(M). Then

GM⁢f⁢(x)=M⁢(f⁢(x+1M)-f⁢(x))-1M⁢f′⁢(x)

This is discussed in more detail on Day 15. As M→∞, GM⁢f⁢(x)→12⁢f′′⁢(x). Therefore,

(X(M))t≥0→M→∞𝑑(Bt)t≥0

2 Theory

Let (Xt)t≥0 be a time-homogeneous Markov process on a locally compact, separable metric space S, and define C0:=C0⁢(S) to be the set of all continuous functions f:S→ℝ such that f⁢(x)→0 as |x|→∞. Note that C0 is a Banach space with norm ||⋅||∞. Assume ℙ({Xt∈S})=1 for all t.

Definition 2.1.

Define the transition semigroup (Pt)t≥0 by (Pt⁢f)⁢(x)=𝔼x⁢[f⁢(Xt)] for f∈C0.

Note.

The assumption

(Xt|X0=x)→x→y𝑑(Xt|X0=y) (4)

implies Pt:C0→C0.

We have the following properties:

  1. (1)

    P0=I since P0⁢f⁢(x)=𝔼x⁢[f⁢(X0)]=f⁢(x).

  2. (2)

    Ps⁢Pt=Ps+t since

    Ps⁢Pt⁢f⁢(x)=𝔼x⁢[Pt⁢f⁢(xs)]=𝔼x⁢[𝔼xs⁢[f⁢(xt)]]=𝔼x⁢[f⁢(xt+s)]=Ps+t⁢f⁢(x)

    If we assume that

    (Xt|X0=x)→t→0𝑑x (5)

    for each x then

  3. (3)

    Pt→I as t→0.

Definition 2.2.

The generator of (Xt)t≥0 and/or (Pt)t≥0 is

G=limϵ→0⁡1ϵ⁢(Pϵ-I)

and

Pt=et⁢G=∑n≥0tnn!⁢Gn (6)

if the above statements make sense.

Note.

If (Xt)t≥0 satisfies (4) and (5) it is Feller. The generator of a Feller process uniquely determines its distribution. This is clear from (6) when (6) makes sense. To prove this claim in general, use resolvents.

The generator has the following properties:

  1. (1)

    G⁢1=0 since (Pt-I)⁢1⁢(x)=𝔼x⁢[1-1]=0

  2. (2)

    π is a stationary measure for (Xt)t≥0 if and only if

    ∫G⁢f⁢(x)⁢π⁢(d⁢x)=0

    for all f.

Next we consider some examples.

Example 2.3.

Let G⁢f⁢(x)=f′⁢(x). Then

Pt⁢f⁢(x)=∑n≥0tnn!⁢f(n)⁢(x)=∑n≥0tnn!⁢f(n)⁢(x+t-t)=f⁢(x+t)

So Xt=X0+t. Therefore, dd⁢x corresponds to “deterministic flow at rate 1.”

Example 2.4.

Let G⁢f⁢(x)=12⁢f′′⁢(x). Recall that this is the generator for Brownian motion. Then

Pt⁢f⁢(x)=∑n≥02-n⁢tnn!⁢f(2⁢n)⁢(x)

Let f^⁢(ξ) be the Fourier transform of f. Then

Pt⁢f^⁢(x)=∑n≥02-n⁢tnn!⁢(-ξ2)n⁢f^⁢(ξ)=e-t2⁢ξ2⁢f^⁢(ξ)

Because e-t2⁢ξ2 is the Fourier transform of the Gaussian density with variance t, and the Fourier transform takes convolution to multiplication,

Pt⁢f⁢(x)=∑n≥02-n⁢tnn!⁢f(2⁢n)⁢(x)=∫-∞∞12⁢π⁢t⁢e-(y-x)22⁢t⁢f⁢(y)⁢𝑑y

It turns out that if G⁢f⁢(x)=f(k)⁢(x) for k>2, G is not the generator of a Feller process. One reason for this is that for k>2 there is no way to write a discrete approximation to f(k)⁢(x) as a sum over values of f with all coefficients except that of f⁢(x) positive. More generally, we can appeal to the following theorem.

Theorem 2.5 (Hille-Yosida Theorem).

Let A be a linear operator on 𝒟⊆C0. Then A has closure that is the generator of a Feller process if and only if

  1. (i)

    𝒟 is dense in C0

  2. (ii)

    The range of λ-A is dense in C0 for some λ>0.

  3. (iii)

    (Positive Maximum Principle) If f⁢(x)≤f⁢(x0) for all x∈S and f⁢(x0)>0, f∈C0, then A⁢f⁢(x0)≤0.

Note.

If (i)-(iii) hold, then (λ-A)-1 exists for all λ>0. To see this, suppose that λ-A is not invertible on C0 for some λ>0. Then there exists f∈C0 such that A⁢f=λ⁢f. Then 𝔼x⁢[f⁢(Xt)]=et⁢λ⁢f⁢(x). But 𝔼x⁢[f⁢(Xt)] is bounded since f∈C0, so we have reached a contradiction.

Note.

If A=limt→0⁡1t⁢(Pt-I), then (iii) holds. For if

A⁢f⁢(x0)=limt→0⁡1t⁢(Pt-I)⁢f⁢(x0)=limt→0⁡1t⁢𝔼x0⁢[f⁢(xt)-f⁢(x0)]

Since f⁢(xt)-f⁢(x0)≤0 by assumption, 𝔼x0⁢[f⁢(xt)-f⁢(x0)]≤0, so A⁢f⁢(x0)≤0.
